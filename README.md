# 🚦 Traffic Monitoring System

A real-time AI-based traffic monitoring system using Computer Vision and Flask.

## 🔥 Features

- 🎥 Real-time video processing
- 🚗 Vehicle detection (YOLO / Stub mode)
- 📊 Traffic density calculation
- 🚦 Congestion prediction (FREE → GRIDLOCK)
- 📡 Live updates using SSE (Server-Sent Events)
- 📈 Dashboard with charts
- ⚠️ Alerts for heavy traffic
- 📁 CSV export of session data
- 🎬 MJPEG live video stream

---

## 🧠 Tech Stack

- Python
- Flask
- OpenCV
- NumPy
- Ultralytics (YOLO)
- Threading & Queues
